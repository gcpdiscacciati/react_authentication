<h1>react_authentication</h1>
<p>Projeto simples de autenticação usando React, React Native, MongoDB e Node JS</p>

<h2>Uso</h2>
<p>Para ativar a API de autenticação, deve-se inicialmente executar o Mongo Shell e, em seguida, executar o Node no terminal com:</p>
<code>node API/src/index.js</code>
<p>A versão do node utilizada foi a 12.13.0</p>

<p>Com a API funcionando, pode-se utilizar a versão Web (React) ou Mobile (React Native) para autenticar um usuário</p>

<h2>Web (React)</h2>
<h3>Uso</h3>
<code>yarn start</code>
<h3>Bibliotecas</h3>
<ul>
  <li>axios</li>
  <li>react-dom</li>
  <li>react-router-dom</li>
  <li>styled-components</li>
</ul>

<h2>Mobile (React Native)</h2>
<h3>Uso</h3>
<p>Com o emulador sendo executado, digite no terminal:</p>
<code>react-native run-android</code>
<h3>Bibliotecas</h3>
<ul>
  <li>axios</li>
  <li>@react-navigation/native</li>
  <li>@react-navigation/native-stack</li>
  <li>react-native-screens</li>
  <li>react-native-safe-area-context</li>
</ul>
